# Mono Odometry

Simply visual odometry using for tracking moving objects, such as *robots, drones, etc.* This program shows the basic usage of Mono Odometry. It track a moving object and get it's trajectory.



## 使用自己的数据集进行姿态估计

## Run
```
cd ~/mono_odometry
mkdir build
cd build
cmake ..
make
./mono_vo ~/test/images
```
